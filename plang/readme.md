# Personal Language
A shortened language dialect that compiles down to C.
That way I can have a less terse language, but with
all the control.

## Examples

### for loop
for(int i = 0; i < size; i++) {
	/* do stuff */
}

for i :size {
	/* do stuff */
}

for i 0:1:size {
	/* do stuff */
}

/* generic */
for index_identifier start:step:end {
	/* do stuff */
}
